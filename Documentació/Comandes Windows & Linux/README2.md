# Comandes DOS

**Prova cada comanda i digues què fa**

```console
FORMAT    //Formateig de particions
CHKDSK    //Comprova sistema de archius en busca d’errors
DISKCOPY  //Copia contingut disquet
DISKCOMP  //Compara contingut de dos disquets
LABEL     //Configuracio etiquetes de volums del disk
VOL       //Mostra etiqueta de volum de disk i nº serie
DEFRAG    //Defragmenta documents de la partició
SCANDISK  //Comprova integritat d'arxius i disc
PROMPT    //Especifica nou simbols sistema de cmd.exe
CLS       //Neteja la finestra
VER       //Mostra la versió de Windows
DATE      //Mostra data
TIME      //Mostra el temps(hora, minuts..)
ECHO      //Mostra missatges
EXIT      //Tanca la terminal
